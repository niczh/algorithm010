## L16 位运算
- 十进制 -- 二进制 转换
- 位运算符：  
    - 左移  <<  新增的补零，移出的丢弃
    - 右移  >>  新增的补零，移出的丢弃
    - 按位或    |
    - 按位与    &
    - 按位取反  ~
    - 按位异或(相同为零不同为一)  ^
- XOR 异或
    - 不进位加法
    - 一些特点：  
        (1s = ~0)  
        x ^ 0 = x  
        x ^ 1s = ~x  
        x ^ (~x) = 1s  
        x ^ x = 0  
        c = a ^ b => a ^ c = b, b ^ c = a  //交换两个数  
        a ^ b ^ c = a ^ (b ^ c) = (a ^ b) ^ c //associative
- 指定位置的位运算
- 判断奇偶：(位运算比模操作快)
    - x % 2 == 1 --> (x & 1) == 1
    - x % 2 == 0 --> (x & 1) == 0
- x >> 1 --> x / 2
- X = X & (X - 1) 清零最低位的1
- X & -X --> 得到最低位的1
- X & ~X --> 0
- 通过位运算改变字符大小写   
    - 使用或运算 | 和空格，实现英文字符转换成小写：  
    ('a' | ' ') = 'a'  
    ('a' | ' ') = 'a'  
    - 使用于运算 & 和下划线，实现英文字符转换成大写：  
    ('a' & '_') = 'A'  
    ('A' & '_') = 'A'  
    - 使用异或运算 ^ 和空格，实现英文字符大小写互换：
    ('a’ ^ ' ') = 'A'
    ('A' ^ ' ') = 'a'
    - 判断两个数是否异号：  
    (x ^ y) < 0 ? # true:异号；false:同号  
    通过补码的符号位，简洁且避免乘除运算时的边界判断。
    - 交换两个数：
    ```
    int a = 1, b = 2;
    a ^= b;
    b ^= a;
    a ^= b;
    // 此时 a = 2, b = 1
    ```

## L17 布隆过滤器和LRU缓存
### 布隆过滤器 Bloom Filter
- 定义：  
    1. 实现一个超大的位数组（二进制向量），和N个哈希函数。
    2. 将位数组先全部置零；
    3. 根据哈希函数个数，假设为N，将一个元素通过N个哈希函数映射产生N个哈希值，分别对应位数组上的N个点，将对应位置标记为1。以此表示该元素存在于此集合；
    4. 查询元素时，同样将元素通过哈希映射到N个点上，若位数组中对应的N个位置存在点不为1，则可判断元素一定不存在于集合中。  

- 特征：  
    - 类似Hash Table，区别在于Hash Table一般还存储其他信息，而布隆过滤器只检索是否存在。
    - 一个很长的**二进制**向量和一系列**随机映射函数**。
    - 可以用于检索一个元素是否在一个集合中
    - 优点：空间效率和查询时间远超一般算法
    - 缺点：有一定的误识别率、删除困难
    - 查询结果：结果不存在则一定不存在，结果存在不一定存在。
    - **一般用于外部初步查询判断。**

### LRU Cache
- LRU: Least Recently Used
- 特征：
    1. 元素有时序特征，区分“最近使用”程度，容量满后删除最久未使用的元素。
    2. 需要快速查找key的元素。
    3. 访问某个key元素时，需要将该元素变为“最近使用”。
- 核心数据结构为哈希表+双向链表。哈希表完成快速查找的部分，双向链表处理快速插入/删除元素的部分。
- get 和 put 方法的时间复杂度为 O(1)
- 一些细节：
    - 双向链表中需要同时存储key和value，而不是只存value。因为在删除最久未使用元素时，需要通过key去Hash表同步删除key-node元素。
    - 很多语言都有内置的哈希链表或者雷瑟LRU功能的库可以使用，例如：Java的LinkedHaMap、Python的OrderedDict等

## L18 排序算法
分类：两类排序算法：  
1. 比较类排序：  
    通过比较来决定元素间的相对次序，时间复杂度不能突破O(nlogn)，也称为非线性时间比较类排序。  
    包括：
    - 交换排序：
        - 冒泡排序
        - 快速排序
    - 插入排序：
        - 简单插入排序
        - 希尔排序
    - 选择排序：
        - 简单选择排序
        - 堆排序
    - 归并排序：
        - 二路归并排序
        - 多路归并排序
2. 非比较类排序：  
    不通过比较类决定元素间的相对次序，以线性时间运行，也称为线性时间非比较类排序。  
    包括：
    - 计数排序
    - 桶排序
    - 基数排序
### 初级排序 - O(N^2)
- 选择排序 Selection Sort  
每次找最小值，然后放在待排序数组的起始位置。
    ```
    # 代码模板
    def selectionSort(array):
        length = len(array)
        for i in range(length -1):
            mini = i
            for j in range(i+1, length):
                if array[j] < array[mini]:
                    mini = j

            array[i], array[mini] = array[mini], array[i]
        return array
    ```
- 插入排序 Insertion Sort  
从前到后逐步构建有序序列；对于未排序数据，在已排序序列中从后向前扫描，找到相应位置插入。
    ```
    # 代码模板
    def insertionSort(array):
    length = len(array)
    for i in range(1, length):
        curr = array[i]
        j = i - 1
        while j >= 0 and array[j] > curr:
            array[j + 1] = array[j]
            j -= 1
        array[j + 1] = curr

    return array
    ```
- 冒泡排序 Bubble Sort  
两层嵌套循环，每次查看相邻的元素，如果逆序则交换。
    ```
    # 代码模板
    def bubbleSort(array):
        length = len(array)
        for i in range(length - 1):
            for j in range(length - 1 - i):
                if array[j] > array[j+1]:
                    array[j], array[j+1] = array[j+1], array[j]

        return array
    ```
### 高级排序 - O(NLogN)
- 快速排序 Quick Sort  
取标杆pivot，将小元素放在pivot左边，大元素放右侧，然后依次对左边和右边的子数组继续快排，以达到整个集合有序。  
- 归并排序 Merge Sort  （分治）  
1.把长度为n的序列分成两个长度为n/2的自序列；  
2.对这两个自序列分别采用归并排序；  
3.将两个排序好的自序列合并成一个最终的排序序列。  
**需要额外空间**
- 归并 和 快排 有相似性，但步骤相反：  
归并：先排序左右数组，在合并两个有序子数组  
快排：先调配出左右子数组（先正确放置中间元素pivot），再对于左右子数组进行排序。  
- 堆排序 Heap Sort  
插入：O(logn)，取最大/小值：O(1)  
1.所有元素依次建立小顶堆；
2.依次取堆顶元素，并删除。依次取出的元素形成有序集合。
### 特殊排序 - O(N)
- 计数排序 Counting Sort
    **要求输入数据是有确定范围的整数**
    1. 找出最大、最小元素；
    2. 遍历所有元素，统计元素值出现的次数
    - 时间复杂度 O(n + k)
    - 空间复杂度 O(n + k)
- 桶排序
    - 时间复杂度 O(n + k)
    - 空间复杂度 O(n + k)
- 基数排序
    - 时间复杂度 O(n * k)
    - 空间复杂度 O(n + k)
